    <h2>Installation</h2>
    <ol>

        <li><strong>Add</strong> <code>ensure df_nocardespawn</code> to your <code>server.cfg</code> file.</li>
        <li><strong>Configure</strong> the parameters according to your needs in the script's configuration file.</li>
